Скелет "Портфель задач". Пример построчного параллельного умножения матриц

debug - логическая отладка (по умолчанию)
emulwin - дискретно-событийное имитационное выполнение для предсказания ускорения
mpi - распределённые вычисления с использованием библиотеки MPI
posix - параллельные вычисления в общей (разделяемой) памяти с использованием API POSIX threads
windows - параллельные вычисления в общей (разделяемой) памяти с использованием Windows API
